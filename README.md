# ✨ Starlight Letter

**별빛 속에서 AI 캐릭터와 편지를 주고받는 데스크톱 위젯**

밤하늘의 별처럼 반짝이는 편지함에서 특별한 캐릭터와 따뜻한 편지를 나눠보세요.

---

## 🌙 주요 기능

### 💌 편지 교환
- 하루에 한 번, AI 캐릭터에게 편지를 보내세요
- 다음 날 별빛처럼 따뜻한 답장이 도착합니다
- 캐릭터의 성격과 말투를 자유롭게 설정할 수 있어요

### 🔮 오늘의 운세
- 캐릭터가 직접 전해주는 오늘의 운세
- 행운의 색상, 피해야 할 것, 응원 메시지까지

### 📅 편지 보관함
- 달력으로 주고받은 편지를 한눈에
- 우표가 찍힌 예쁜 편지지 형태로 보관
- PNG 이미지로 저장 가능

---

## 🎨 디자인 컨셉

- **밤하늘 테마**: 보라색과 파란색의 우주적인 그라데이션
- **별 반짝임**: 자연스러운 별빛 애니메이션
- **글로우 효과**: 은은하게 빛나는 UI 요소들

---

## ⚙️ 설정 옵션

- **API Key**: Gemini 2.5 Pro API 키
- **내 이름**: 캐릭터가 부를 이름
- **내 소개**: 나에 대한 설명 (선택)
- **캐릭터 프롬프트**: 캐릭터의 성격, 말투, 특징
- **관계 설정**: 친구, 선배, 연인 등
- **말투**: 존댓말 / 반말
- **폰트 선택**: 6가지 한글 폰트 지원

---

## 🚀 사용 방법

1. 위젯을 클릭하면 편지 메뉴가 열립니다
2. 우클릭으로 컨텍스트 메뉴 사용 가능
3. 설정에서 API 키와 캐릭터를 설정하세요
4. 매일 편지를 쓰고, 다음 날 답장을 확인하세요!

---

## 📁 데이터 저장 위치

- Windows: `%APPDATA%/StarlightLetter/`
- 편지, 운세, 설정이 JSON 형식으로 저장됩니다

---

## 🛠️ 기술 스택

- **Electron** - 데스크톱 앱 프레임워크
- **Gemini 2.5 Pro** - AI 답장 생성
- **html2canvas** - 편지 이미지 저장
